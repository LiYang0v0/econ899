# Data for analysis

This folder will contain all data for analysis.

- The contents of this folder are generated by the code in `/programs/01_dataprep`
  from the raw data in `/data/raw_data`.
- The contents of this folder will be used by the code in `/programs/02_analysis`
  to generate the results in `/results`.
- Use your `.gitignore` file to exclude the contents of this folder (other than
  this README) from the replication package.
