# Results folder

This folder will contain the results of your analysis.

- Ideally, each table and figure should be output to a separate file, exactly as
  it appears in your text.
- These files should be generated by your analysis code, and should not be
  included in the replication package itself.
- The files should be given names that correspond to their role in the paper.
  For example, `table01.tex` would contain the results reported in Table 1.
- Use your `.gitignore` file to exclude results output from your GitHub repo.
